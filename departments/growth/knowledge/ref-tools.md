# Growth, Sales & New Markets Tools

Daily tools and integrations used across the Growth, Sales, and New Markets teams.

## Primary Tools

| Tool | Purpose | Primary Users | Frequency |
|------|---------|--------------|-----------|
| Salesforce | CRM — pipeline, prospects, deal tracking, account records | Sales, Revenue Ops | Daily |
| HubSpot | Marketing automation, lead generation, campaigns | Growth (Alex, Jared) | Daily |
| SharePoint Online | Document collaboration — proposals, RFPs, board decks, state docs | All teams | Daily |
| Outlook / Office 365 | Email, calendar, scheduling | All teams | Daily |
| Slack | Team communication, prospect channels, cross-dept coordination | All teams | Daily |
| Confluence | Strategic planning docs, EBT design principles, GTM strategy | New Markets | Weekly |
| Zoom | Video meetings — prospect calls, state government meetings | All teams | Weekly |

## Document Patterns

New Markets is a document-heavy team. Key document types and where they live:

| Document Type | Location | Examples |
|--------------|----------|---------|
| Government RFP/RFI responses | SharePoint | City of Boston RFI, CareSource RFQ |
| Federal proposals (FNS/USDA) | SharePoint | FNS Draft Smart Benefits, USDA One-Pager, EBT Capabilities Overview |
| State implementation materials | SharePoint | Iowa Implementation Questions, Nebraska Meeting docs, Idaho SOW |
| Board meeting decks | SharePoint | Monthly board meeting presentations |
| OKR tracking | SharePoint (Excel) | Commercial OKRs weekly tracker |
| EBT strategic planning | Confluence | EBT GTM Strategy, EBT Design/Build Principles, EBT Phased Launches |
| Competitor analysis | SharePoint | Conduent Item-Level Restrictions |
| Sales collateral | SharePoint | EBT capabilities decks, state-specific presentations |

## Integrations

- **Salesforce + HubSpot**: Marketing-sourced leads flow from HubSpot into Salesforce for pipeline tracking by Sales and Revenue Ops.
- **SharePoint + Office 365**: Documents are created and collaboratively edited in SharePoint, shared via Outlook email and Teams.
- **Confluence + Jira**: Strategic planning happens in Confluence; any resulting execution tasks are tracked in Jira (though New Markets operates primarily through documents, not tickets).
- **Slack + All Tools**: Prospect-specific Slack channels (`#prospect-*`) serve as the coordination layer across all tools for active deals and state engagements.
