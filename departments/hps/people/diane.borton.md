# Diane Borton

**Title**: Senior Manager, Health Plan Solutions

**Department**: Health Plan Solutions (HPS) -- Implementation

**Reports to**: Ariane Grazian (SVP, Health Plan Solutions)

## Role

Diane is an Implementation Manager (IM) responsible for managing customer-facing implementation calls, OnRamp module management, and materials coordination for her assigned health plan sponsors.

## Key Responsibilities

- Managing end-to-end implementation workflows for assigned customers
- Running customer-facing implementation calls and managing timelines
- OnRamp module management -- ensuring sponsors complete their required forms and submissions
- Materials coordination -- working with MX on card packages, catalogs, and digital content
- Eligibility file testing sign-off coordination
- Coordinating across internal teams (Engineering, MX, Care, Finance) for each customer

## Customer Portfolio

- **SCAN Health Plan (SCN)**: Multi-brand (SCNWA for Washington, SCNCA for California R&I). Key issues include eligibility file address mapping, SFTP encryption/naming, material carrier reprints, and multi-language IVR.
- **SummaCare (SMC)**: Ohio-based, ~30,000 projected cards. Products include OTC, Vision, and Rewards. Key issues include eligibility file support, funding account setup, and card timing vs. call center readiness.
